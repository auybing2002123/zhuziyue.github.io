# 图片资源目录

## 目录结构

- `hero/` - Hero轮播图背景图片
- `stats/` - 统计数据区域背景图片  
- `footer/` - 底部信息背景图片
- `default/` - 默认占位图片

## 使用说明

1. **默认图片**：项目提供默认的静态图片
2. **用户上传**：用户可以通过编辑功能上传自定义图片
3. **存储机制**：用户上传的图片存储在浏览器localStorage中
4. **回退机制**：如果localStorage中没有图片，则使用默认静态图片

## 图片规格建议

- **Hero轮播图**：1920x1080 (16:9)
- **统计区域背景**：1920x600
- **底部背景**：1920x400
- **文件格式**：JPG/PNG
- **文件大小**：建议每张不超过500KB
